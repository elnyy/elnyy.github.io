【まどドラ】必殺技名で覚える英語 (アタッカー編)

まどドラの必殺技名(special)の英語版を見ていくことで英語を少しでも覚えてみようという試みです。

一気に全キャラの必殺技名を調べようとすると文字量が膨大になってしまうので、今回はアタッカー編とします。

暁美ほむら - ミサイルによる集中砲火#

暁美ほむら「ミサイルによる集中砲火」は Concentrated Missile Fire です。

concentratedって初めて聞きました。あと「ミサイル」が英語だということは何となく感じていましたがMissileと綴ることは初めて知りました。

単語の意味↓

  • Concentrated(adjective) … 集中した、濃縮した、濃厚な
  • Missile(noun) … ミサイル、飛び道具
  • Fire(noun) … 火、火炎、(暖房・料理用の)火、炉火、炭火、たき火、かがり火、暖房器、ヒーター、火事

七海やちよ - アブソリュート・レイン#

七海やちよ「アブソリュート・レイン」は Absolute Rain です。

日本語の発音をそのまま英語にしているので覚えやすそうですね。

単語の意味↓

  • Absolute(adjective) … (比較・相対を離れて)絶対の、絶対的な、完全な、まったくの、純粋の、無条件の、制約のない、疑問の余地のない、明白な、確かな
  • Rain(noun) … 雨、降雨、雨天、(熱帯地方の)雨季、(…の)雨

美国織莉子 - オラクルレイ#

美国織莉子「オラクルレイ」は Oracle Ray です。

これまた、読み方そのままの英語です。Rayはx-ray(x線)と同じrayかな…?

単語の意味↓

  • Oracle(noun) … (古代ギリシャの)神託、託宣、神託所、神命、(エルサレム神殿内の)至聖所、(古代ギリシャの)神託を伝える人
  • Ray(noun) … 光線、わずか、少量、熱線、放射線、輻射(ふくしや)線

佐倉京子 - 盟神抉槍#

スクショのタイミングでちょうど目を瞑っていた...！

佐倉京子「盟神抉槍」は Kugatachi です。

ローマ字表記にしただけなので、英語要素はありませんが,,,「盟神抉槍」の読み方をいつも忘れるのでこの機会に覚えておきたいです

十咎ももこ - エッジオブユニヴァース#

十咎ももこ「エッジオブユニヴァース」は The Universe’s Edge です。

The Univsersee’s Edgeそのままだと日本人には読みずらいからかな？

日本語名も英語名も、意味は同じですが表現方法を少し変えていますね。これは覚えずらいポイント+1です(?)

単語の意味↓

  • Universe(noun) … 宇宙、天地万有、万物、全人類、(人間の活動の場としての)世界、領域、分野
  • Edge(noun) … (刃物の)刃、(刃の)鋭利さ、鋭さ、(欲望・言葉などの)激しさ、痛烈、強み、優勢、(二つの線の接する)縁、へり、かど

アルティメット・まどか - もう絶望する必要なんてない！#

アルティメット・まどか「もう絶望する必要なんてない！」は Nothing to Despair, Ever です。

このセリフ聞くたびにしびれます。翻訳もシンプルで覚えやすそうです。

単語の意味↓

  • Nothing(pronoun) … 何も…ない、少しも…ない
  • Despair(noun) … 絶望、絶望させるもの
  • ever(adverb) … かつて、いつか、これまでに、かつて(…することがない)、決して(…ない)、いずれ、これまで、今まで、いつも、常に
